取消自动生成getter()和setter()

通常，Kotlin会自动为类中未定义getter()setter()的属性自动生成默认的getter()和setter():

1
2
3

class Foo{
    var abc : String? = null
}

Kotlin 编译器会自动将属性补全为以下形式：

1
2
3

    private String abc;
    public final String getAbc(){...}
    public final void setAbc(String abc){...}

如果属性不需要getter()或setter(), 或者需要禁用生成的getter()或setter()以便构造Java模式的Builder，我们可以使用注解@JvmFile
